## 5. UTM Parameters

### 5.1 UTM Structure

```
utm_source=google
utm_medium=cpc
utm_campaign={campaign_name}
utm_content={ad_group}
utm_term={keyword}
```

### 5.2 Example URLs

**Google SWE:**
```
https://togtog.com/prep/google-software-engineer?utm_source=google&utm_medium=cpc&utm_campaign=tech-swe&utm_content=google-swe&utm_term=google+software+engineer+interview
```

**McKinsey:**
```
https://togtog.com/prep/mckinsey-consultant?utm_source=google&utm_medium=cpc&utm_campaign=consulting&utm_content=mckinsey&utm_term=mckinsey+case+interview

Use Google Ads ValueTrack parameters for automatic population:
```

utm_medium=cpc
utm_campaign={campaignid}
utm_content={adgroupid}
utm_term={keyword}

## 7. Conversion Tracking

### 7.1 Conversion Events

| Event | Type | Value | Attribution |
|-------|------|-------|-------------|
| Purchase | Primary | Dynamic ($197) | Data-driven |
| Add to Cart | Secondary | $20 | Data-driven |
| View Pricing | Micro | $5 | Data-driven |
| Start Checkout | Secondary | $30 | Data-driven |
| Email Signup | Micro | $10 | Data-driven |

### 7.2 Implementation

**Google Tag Manager Setup:**
```javascript
// Purchase event
gtag('event', 'purchase', {
  'send_to': 'AW-XXXXXXXXX/YYYYYYYYYYYY',
  'value': 197.00,
  'currency': 'USD',
  'transaction_id': '{{order_id}}'
});

// View pricing event
gtag('event', 'view_item', {
  'send_to': 'AW-XXXXXXXXX/ZZZZZZZZZZZZ',
  'value': 5.00,
  'currency': 'USD',
  'items': [{
    'id': '{{position_slug}}',
    'name': '{{position_name}}'
  }]
});
```

### 7.3 Attribution Window

- **Click-through:** 30 days
- **View-through:** 1 day
- **Model:** Data-driven (or Position-based if insufficient data)
