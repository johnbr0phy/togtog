## Leadership Principles for PM Interviews

Each interviewer is assigned 2-3 LPs to assess:

| LP | Description | Interview Focus |
|----|-------------|-----------------|
| Customer Obsession | Start with customer, work backward | Dedicated round |
| Think Big | Bold direction, inspire results | Vision & Strategy |
| Invent and Simplify | Innovation and simplification | Vision & Strategy |
| Dive Deep | Stay connected to details | Execution |
| Bias for Action | Speed matters | Execution |
| Disagree and Commit | Challenge, then commit | Collaboration |
| Earn Trust | Listen, speak candidly | Collaboration |
| Are Right, A Lot | Good judgment | Bar Raiser |
| Ownership | Think long-term | Throughout |

**Most Critical for PMs:**
1. Customer Obsession
2. Bias for Action
3. Ownership
4. Have Backbone; Disagree and Commit
